# DualTrack
***
Dual Track es una aplicación para extraer e insertar pistas de audio combinado en un solo archivo. Te permite agregar 2 videos y exportar uno con ambas pistas, útil cuando tenes varias versión del mismo video en diferentes idiomas y las queres combinar
### Instalación
***
### [Instalacion de FFMPEG](https://github.com/ffmpeg/ffmpeg)
##### Para distros basadas eb Debian:
* sudo apt update
* sudo apt install ffmpeg

### [DualTrack Releases](https://github.com/Facusch808/DualTrack/releases/tag/v1)
* wget https://github.com/Facusch808/DualTrack/releases/download/v1/DualTrack.deb
* cd (en la carpeta donde lo descargaste, generalmente descargas)
* sudo dkpg -i DualTrack.deb
